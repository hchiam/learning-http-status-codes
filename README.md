# Learning HTTP status codes

Just one of the things I'm learning. <https://github.com/hchiam/learning>

- 100+ = **info**
- 200+ = **success**
- 300+ = **redirect**
- 400+ = **client** error ("it's you")
- 500+ = **server** error ("it's not you user, it's me")
